# Sports Science Portfolio

This repository contains applied sports science and performance analytics projects focused on athlete monitoring, workload reporting, dashboard development, and performance data workflows. The projects were built to support coaching staff with faster, clearer, and more practical decision-making.

## Projects

### 1. Daily Training Load Report
An automated daily report that summarizes GPS load and wellness data to help identify athletes who may need closer monitoring.

### 2. Weekly Training Load Report
An automated weekly report that provides an overview of team workload, recovery trends, and periodization support.

### 3. Athlete Performance Dashboard
An interactive Shiny dashboard that brings together workload, wellness, positional comparisons, and athlete-specific trends in one place.

### 4. Web Scraping Project
A Python workflow that collects public performance data and turns it into a clean dataset for analysis and modeling.

## Skills Demonstrated

- Data cleaning and wrangling
- Automated reporting
- Interactive dashboard development
- Performance monitoring and workload analysis
- Web scraping and data preparation
