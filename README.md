# Sports Science Portfolio

This repository contains applied sports science and performance analytics projects focused on athlete monitoring, workload reporting, dashboard development, and performance data workflows. The projects were built to support coach, athlete, stakeholder, and medical staff facing reports for faster, clearer, and more practical decision-making. 

## Projects

### 1. Daily Training Load Report
An automated daily report that summarizes GPS load and wellness data to help identify athletes who may need closer monitoring. This coach-facing report can assist with daily practice planning. 

### 2. Weekly Training Load Report
An automated weekly report that provides an overview of team workload, recovery trends, and periodization support. This coach facing report assist in decision making across the macrocycle.

### 3. Athlete Performance Dashboard
An interactive dashboard built in R Shiny that brings together session-level performance monitoring and athlete comparison tools in one place. This dashboard contains coach and athlete facing tabs to work across a multitude of conversations. 

### 4. Web Scraping Project
A Python workflow that collects public performance data and turns it into a clean dataset for analysis and modeling. 

### 5. High Speed Running Management Report
A HSR dashboard that filters by session type and date, summarizes positional high-speed running, and flags athletes based on distance from same-day positional means. This dashboard was created for coach-facing decision making in regard to HSR supplementation. 

### 6. Dynamic HTML Athlete Monitoring Report
A dynamic browser-based athlete monitoring report built with R, HTML, CSS, JavaScript, and Chart.js. This project uses an R script to read Excel-based monthly, weekly, and schedule data, convert them into JSON, and inject them into an HTML template that displays interactive timelines, ROM comparisons, strength leaderboards, mobility trends, and schedule-linked monitoring visuals. This type of visual is effective for coaches, athletes, stakeholders, or medical staff to support red flagging athletes as well as continual monitoring in a dynamic, yet simple format.
